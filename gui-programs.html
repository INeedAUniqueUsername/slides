Introduction

  • GUI programming sucks
  • There are A LOT of choices
  • Most of them suck

Terminal

  • Console.WriteLine + Console.ReadLine everywhere
  • Simple
  • Doesn't look very good
  • Not fun to use
  • Lots of string handling

Java Swing

  • Very finicky
  • Cross-platform (maybe)
    • Works on my machine (lol)
  • Doesn't scale easily
  • Looks like Java
  • Have to deal with Java

tkinter (Python)

  • Never used it before
  • Name doesn't sound very appealing
  • Not sure I want to use Python

HTML/CSS/JS

  • Basically cross-platform
  • All text is selectable
  • Anything can have click events
  • Can change the page by setting strings
  • Uses a lot of CPU
  • Have to deal with HTML/CSS/JS

Windows Forms or Windows Presentation Foundation

  • Definitely not cross-platform
