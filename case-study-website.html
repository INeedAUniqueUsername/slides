HTML

  • <head> contains title, link, style, etc
    • <title> shows on tab (not in page)
    • <link> loads CSS, font, etc
    • <style> loads CSS from text
  • <body> contains the actual content

HTML Body

  • Elements
    • <div> - general container
    • <a> - link
    • <h1>, <h2>, <h3>... - headings
    • <p> - paragraph (large vertical margin)
    • <input> - text input
    • <script> - JS code
  • Created in order defined

  • Has tag and attributes
  • May have inner text or HTML

HTML Attributes

  • id
    • Identifies one particular element by name
    • Also works as an anchor
  • class
    • Apply styles from CSS classes
    • Can have multiple (applied in order listed)
  • style
    • Contains inline CSS code (applied last)

HTML Events

  • onclick=: fires when clicked - very useful
  • onclick="foo(this)" works (passes in this element)
  • Mouse: onmouseover, onmouseout, etc
  • Other: onchange (input), onselect (text selected), etc
  • ALL elements have these events - very useful

JavaScript (1)

  • Global variables: window, document
  • document.getElementById(id): Very important function
  • document.body.querySelector(cssSelector)
  • document.body.querySelectorAll(cssSelector)
  • Can set events on elements via code
  • Event Handler: element.onclick = () => { ... };>
  • Event Listener: element.addEventListener("click", () => { ... })>

JavaScript (2)

  • document.createElement(tag) to create element object
  • element.id to set id
  • element.className to set class
  • element.style to set inline style (non-class)
    • Some attributes have different names in code
  • element.remove() removes element from its parent
  • element.append(el) appends a new child element

JavaScript (3)

  • Can set HTML strings directly
  • element.outerHTML: Element and its descendants
  • element.innerHTML: Subelements only
  • element.innerText: Text content only
  • element.value: Text value of input/select
  • Changing HTML causes "reflow" - can be expensive

Margin and padding

  • Space between elements
  • Margin: Space outside of border
  • Padding: Space within border
  • Add spacing, but not too much

Display

  • display is a useful attribute
  • block top to bottom (default)
  • inline-block for left to right
  • inline for within text
    • Does not respect width/height

Float

  • float is a useful attribute
  • left top to bottom (default)
  • clear:both on next element to clear
  • Kinda finicky to use

CSS Units

  • Attributes like size= have units
  • Absolute: px (pixel), pt (point)
  • Relative: % (parent element), em (font-size)
  • There are A LOT of units

CSS Animations

  • They're called @keyframes
  • Define properties at each keyframe
  • Interpolates properties between frames
  • Can work with most properties
  • Can define using time percentages 0% - 100%
  • Can define using from and to
  • Apply to an element via its style

This text fades in and our

@keyframes fadein {
    0% { opacity: 0; }
    25% {   opacity: 1; }
    50% { opacity:0.5; }
    75% { opacity: 0; }
    100% { opacity:0.5; }
}

.fading {
    animation-name: fadein;
    animation-duration: 1.5s;
    animation-timing-function: ease;
    animation-fill-mode: both;
    animation-iteration-count: infinite;
}
