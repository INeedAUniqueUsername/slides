Introduction

HTML

  • <head> contains title, link, style
    • <title> shows on tab (not in page)
    • <link> loads CSS, font, etc
    • <link> loads CSS from text
  • <body> contains the actual content

HTML Body

  • Elements
    • <div> - general container
    • <a> - link
    • <h1>, <h2>, <h3>... - headings
    • <p> - paragraph (large vertical margin)
    • <input> - text input
    • <script> - JS code
  • Created in order defined

HTML Attributes

  • id
    • Identifies one particular element by name
    • Also works as an anchor
  • style
    • Contains inline CSS code
  • class
    • Apply styles from CSS classes
    • Can have multiple classes (in order listed)

CSS Styles (1)

  • Contained in <style>, style=, or .css file
  • Each entry begins with selector
    • All tags *
    • Element tag h1
    • class .elementClass
    • id #elementId

CSS Styles (2)

  • Additional selectors
    • Pseudo-class .elementClass:hover
    • Pseudo-element .elementClass::selection
    • All descendants.elementClass p
    • Immediate children .elementClass > p
  • There are A LOT of selectors

CSS Attributes

  • font-style, font-size, font-family
  • border-style, border-size, border-color
  • margin, padding
  • color, background-color
  • width, height
  • text-align, vertical-align (RIP horizontal alignment)
  • There are A LOT of attributes

CSS Units

  • Attributes like size= have units
  • Absolute: px (pixel), pt (point)
  • Relative: % (parent element), em (font-size)
  • There are A LOT of units

CSS Animations

  • They're called @keyframes
@keyframes fadein {
    from { opacity: 0; }
    to {   opacity: 1; }
}

#logo {
    animation-name: fadein;
    animation-duration: 0.8s;
    animation-timing-function: ease-in;
    animation-fill-mode: both;
}
